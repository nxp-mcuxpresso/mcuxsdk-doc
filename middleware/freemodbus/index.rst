FreeModbus
#############

FreeModbus is an open source Modbus stack used to tcp/rtu function development. It provides a modbus library for easy development.
For more information on FreeModbus, see [https://github.com/RT-Thread-packages/freemodbus] [https://github.com/cwalter-at/freemodbus/tree/master/modbus].
