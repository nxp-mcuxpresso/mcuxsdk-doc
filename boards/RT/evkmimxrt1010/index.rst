MIMXRT1010-EVK
####################

Overview
********

The NXP MIMXRT1010-EVK is a development board for the i.MX MIMXRT1011 500 MHz 32-bit ARM Cortex-M7 MCUs.



.. image:: ./evkmimxrt1010.png
   :width: 240px
   :align: center
   :alt: EVK-MIMXRT1010

MCU device and part on board is shown below:

 - Device: MIMXRT1011
 - PartNumber: MIMXRT1011DAE5A
